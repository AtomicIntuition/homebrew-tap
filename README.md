# Homebrew Tap for 3RROR_K1NG

Official Homebrew tap for [3RROR_K1NG](https://3rrork1ng.com) CLI.

## Installation

```bash
brew tap AtomicIntuition/tap
brew install error-king
```

## Usage

```bash
3rror scan https://example.com
```
