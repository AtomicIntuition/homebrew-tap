# Homebrew Tap for Crisp

Official Homebrew tap for [Crisp](https://crisp.sh) CLI.

## Installation

```bash
brew tap AtomicIntuition/tap
brew install crisp
```

## Usage

```bash
crisp scan https://example.com
```
